# OOP Coffee Machine

A Python-based coffee machine simulation using Object-Oriented Programming principles.

## Description

This project simulates a coffee machine that can make different beverages (espresso, latte, cappuccino). It manages resources, processes payments, and tracks profits using OOP concepts.
